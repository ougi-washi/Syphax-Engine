# Utilities

Use this page when the feature you are building depends on shared helpers rather than on a larger runtime system like scene, UI, or physics.

## Start here

- Use [Path: Utilities](../path/utilities.md) if you want the guided walkthrough.
- Use [Start Here: Error handling](../start-here/error-handling.md) if the immediate need is results and validation.
- Use [Start Here: Resource paths](../start-here/resource-paths.md) if the immediate need is packaged or writable paths.

- Resource macros and last-error handling:

- Runtime feature gates:

- Raw quad helpers for custom draw paths:
